== Goals

* Select user persona (WYSIWYG / WYSIWYGM / WYMIWYG)
* Select install strategy (native / container / toolbox / VM)
* Configure path management (_pathroot or modshells)

== Steps

1. **Persona**

4. **Trust Defaults**
